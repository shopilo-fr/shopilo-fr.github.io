Codes promo France

Collection de scripts Python open-source pour la recuperation automatique de codes promo dans les boutiques en ligne francaises. Les donnees sont extraites en temps reel depuis Shopilo.fr, le plus grand agregateur de coupons en France.

Python 3.8+ MIT License npm disponible Donnees live shopilo.fr
# Installer les dependances
pip install requests beautifulsoup4

# Recuperation automatique des codes promo
git clone https://github.com/shopilo-fr/code-promo-[boutique]
python fetch.py
